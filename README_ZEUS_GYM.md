# ⚡ Zeus Gym Dashboard

Modern and comprehensive gym management dashboard built with **React**, **Vite**, and **Tailwind CSS**. Manage members, trainers, schedules, attendance, and more with an intuitive interface.

## ✨ Features

- **Dashboard**: Overview of gym statistics and recent activities
- **Members Management**: Add, view, and manage gym members with membership tracking
- **Trainers**: Manage trainers, specializations, and ratings
- **Class Schedule**: View and manage weekly gym classes with quota tracking
- **Attendance Tracking**: Monitor member attendance with monthly statistics
- **Membership Plans**: Display different membership packages (Basic, Premium, VIP)
- **Announcements**: Post and manage gym announcements
- **User Authentication**: Login and registration system for gym staff
- **Statistics**: Detailed analytics and performance metrics
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices

## 🛠️ Tech Stack

- **Frontend Framework**: React 19.2.5
- **Build Tool**: Vite 8.0.10
- **Styling**: Tailwind CSS 3.4.19
- **Routing**: React Router DOM 7.15.0
- **HTTP Client**: Axios 1.16.0
- **Charts**: Recharts 3.8.1
- **Icons**: React Icons 5.6.0

## 📦 Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd rifqi-zeusgym
```

2. Install dependencies:
```bash
npm install
```

3. Start development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## 🚀 Available Scripts

- `npm run dev` - Start development server with HMR
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally
- `npm run lint` - Run ESLint to check code quality
